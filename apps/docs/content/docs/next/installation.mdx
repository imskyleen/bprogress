## Installation

To install BProgress, run the following command:

<Tabs groupId="packageManager" items={['npm', 'pnpm', 'yarn', 'bun']} persist>
  <Tab value="npm">```bash npm install @bprogress/next ```</Tab>
  <Tab value="pnpm">```bash pnpm add @bprogress/next ```</Tab>
  <Tab value="yarn">```bash yarn add @bprogress/next ```</Tab>
  <Tab value="bun">```bash bun add @bprogress/next ```</Tab>
</Tabs>

Now you are ready to use BProgress in your Next.js application!
